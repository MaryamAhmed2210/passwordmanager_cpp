# 🔐 Password Manager (C++)

A Password Manager developed in **C++** using **Object-Oriented Programming (OOP)** concepts. It allows users to securely manage passwords with authentication, encryption, and file handling.

## Features
- User Registration & Login
- Add, View, Search, Update & Delete Passwords
- Password Encryption & Decryption
- Random Password Generator
- Password Strength Checker
- Activity Logging
- File Handling

## Concepts Used
- OOP
- STL (`vector`)
- File Handling
- Basic Encryption

## Run

```bash
g++ main.cpp Password.cpp PasswordManager.cpp User.cpp Encryption.cpp Logger.cpp CyberTool.cpp -o PasswordManager
./PasswordManager
```
